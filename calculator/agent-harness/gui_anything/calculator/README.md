# Calculator GUI

A cross-platform GUI for the Python Calculator, generated by GUI-Anything.

## Prerequisites

- Node.js 20+
- Python 3.10+ (for the calculator backend)

## Installation

```bash
cd calculator/agent-harness
npm install
```

## Usage

```bash
# Start dev server
npm run dev

# The GUI will be available at http://localhost:5173
```

## Features

- **5 action groups**: Arithmetic, Scientific, Memory, History, Export
- **12 GUI actions** with dynamic forms
- **15 MCP tools** for AI agent control
- **Dark theme** with indigo/purple palette
- **Responsive layout** — works on desktop and mobile

## MCP Agent Integration

The GUI exposes 15 MCP tools via `agent-harness/mcp-tools.json`:

```bash
# Example: Add two numbers
{"tool": "arithmetic__add", "input": {"a": 5, "b": 3}}
# → {"result": 8}

# Example: Take screenshot
{"tool": "take_screenshot", "input": {"path": "calc.png"}}
```

## Architecture

```
gui_anything/calculator/
├── gui-schema.json      # A2UI schema (source of truth)
├── theme.json            # Visual design tokens
├── src/                  # React frontend
├── agent-harness/        # MCP tools + screenshot
├── utils/                # Backend integration
└── tests/                # Validation tests
```
